# wiki-templates/ — LLM Wiki 템플릿 3종 (P2-S2)

> `docs/plan/02-master-plan.md` P2-S2. 강의 STEP 2(요약/비판적 읽기/Literature Matrix 템플릿)를 구약학 연구에 맞게 이식.
> 여기 있는 파일이 원본(SSOT)이다. 실제 배치(Obsidian "LLM WIKI" 볼트로 사본 복사)는 `[IMAC-ONLY]`(P2-S1 선행) — iMac 작업 큐 참조.
> 각 템플릿에 강의 원 구조(문제·방법·데이터·결과·한계 / 주장·근거·약점·재현성 / 비교 매트릭스) + 구약학 변형(본문비평·주석 전통·passages 축)을 반영했다.

- `01-summary-template.md` + `01-summary-sample.md` — 요약 템플릿(문제·방법·데이터·결과·한계 + 전거/본문비평).
- `02-critical-reading-template.md` + `02-critical-reading-sample.md` — 비판적 읽기(주장·근거·약점·재현 가능성 + 주석 전통·본문비평 단서).
- `03-literature-matrix-template.md` + `03-literature-matrix-sample.md` — 논문×축 비교표(Literature Matrix).

배치 시 Obsidian 볼트 구조(P2-S1): `authors/ works/ topics/ passages/ venues/ years/ + index.md + log.md` — `passages`(성경 본문별 축)가 강의 원본 대비 핵심 변형.
